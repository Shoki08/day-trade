# ⚡ デイトレードアドバイザー - 短期投資向け

コインチェック連携のデイトレード・短期投資向けアプリです。リアルタイム分析と高速更新で、短期売買をサポートします。

## 🎯 主な機能

### 1. **高速リアルタイム更新**
- **5秒ごと**に自動更新（長期投資版は1分）
- 価格変動を即座にキャッチ
- リアルタイムで損益を追跡

### 2. **短期テクニカル指標**
- **RSI (14期間)**: 売られすぎ/買われすぎを判定
- **ボリンジャーバンド**: 価格の変動幅を可視化
- **短期移動平均 (5/20)**: 短期トレンドを把握
- すべて短期売買に最適化された設定

### 3. **価格チャート**
- Chart.jsによる滑らかなグラフ表示
- 1分足・5分足・15分足・1時間足対応
- 価格の動きを視覚的に把握

### 4. **板情報（オーダーブック）**
- 買い板・売り板をリアルタイム表示
- 市場の需給バランスを確認
- 約定価格の予測に活用

### 5. **価格アラート**
- 任意の価格でアラート設定
- 目標価格到達時に通知
- 複数アラートの同時設定可能

### 6. **損益計算機**
- エントリー価格と数量を入力
- リアルタイムで損益を自動計算
- 損益率をパーセント表示

### 7. **売買シグナル**
- 複数指標を総合判断
- 買い/売り/様子見を明確表示
- シグナル強度を5段階で表示

## 🚀 使い方

### 基本操作

1. **通貨ペアを選択**
   - BTC/JPY, ETH/JPY, XRP/JPY, LTC/JPY から選択

2. **チャートの時間軸を選択**
   - 1分足: 超短期の動き
   - 5分足: 短期トレンド（推奨）
   - 15分足: 中期トレンド
   - 1時間足: デイトレードの全体像

3. **テクニカル指標を確認**
   - RSI: 30以下で買い検討、70以上で売り検討
   - ボリンジャー: 下限突破で買い、上限突破で売り
   - 移動平均: 5期間が20期間を上回れば上昇トレンド

4. **板情報で需給を確認**
   - 買い板（緑）: 買い注文の厚さ
   - 売り板（赤）: 売り注文の厚さ

5. **価格アラートを設定**
   - エントリーポイントの候補価格を設定
   - 利益確定ラインを設定
   - 損切りラインを設定

6. **損益計算でリスク管理**
   - エントリー価格を入力
   - 取引数量を入力
   - リアルタイムで損益を確認

## ⚡ デイトレード戦略の基本

### エントリー（買い）タイミング
✅ RSI < 30（売られすぎ）  
✅ 価格がボリンジャー下限突破  
✅ 5期間移動平均が20期間を上抜け  
✅ 買い板が厚い

### エグジット（売り）タイミング
✅ RSI > 70（買われすぎ）  
✅ 価格がボリンジャー上限突破  
✅ 5期間移動平均が20期間を下抜け  
✅ 売り板が厚い

### リスク管理
- **損切りライン**: エントリー価格の -3〜5%
- **利益確定**: エントリー価格の +5〜10%
- **ポジションサイズ**: 資金の10〜20%まで
- **同時保有**: 1〜2銘柄まで

## 📊 技術仕様

### 更新頻度
- **価格データ**: 5秒ごと
- **板情報**: 5秒ごと
- **チャート**: リアルタイム更新

### 使用API
- コインチェック Ticker API
- コインチェック Order Book API
- 認証不要（APIキー不要）

### テクニカル指標の計算式

#### RSI (Relative Strength Index)
```
RS = 平均上昇幅 / 平均下落幅
RSI = 100 - (100 / (1 + RS))
期間: 14
```

#### ボリンジャーバンド
```
ミドルバンド = 20期間単純移動平均
上限バンド = ミドルバンド + (標準偏差 × 2)
下限バンド = ミドルバンド - (標準偏差 × 2)
```

#### 移動平均線
```
短期: 5期間単純移動平均
長期: 20期間単純移動平均
```
