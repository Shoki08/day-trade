# 暗号資産アドバイザー - コインチェック連携PWA

投資初心者向けの暗号資産売買サポートアプリです。テクニカル分析に基づいた売買推奨をわかりやすく表示します。

## 🎯 主な機能

### 1. リアルタイム価格表示
- コインチェックのパブリックAPIから最新価格を取得
- 1分ごとに自動更新
- 価格変動率を視覚的に表示

### 2. テクニカル分析
- **RSI (相対力指数)**: 買われすぎ/売られすぎを判断
- **移動平均線**: 短期(7日)・長期(25日)のトレンド分析
- **MACD**: モメンタムの強さを測定

### 3. 売買シグナル
- 🟢 **買い推奨**: 複数の指標が買いシグナルを示している
- 🔴 **売り推奨**: 複数の指標が売りシグナルを示している
- 🔵 **様子見**: 明確なシグナルがない状態

### 4. プッシュ通知
- 重要な売買シグナル発生時に通知
- バックグラウンドでも動作

### 5. PWA対応
- ホーム画面に追加可能
- オフラインでも基本機能が利用可能
- ネイティブアプリのような操作感

## 📱 使い方

### インストール方法

1. **スマートフォンのブラウザで開く**
   - Safari (iOS) または Chrome (Android) で開く
   
2. **ホーム画面に追加**
   - iOS: 共有ボタン → "ホーム画面に追加"
   - Android: メニュー → "ホーム画面に追加"

### 基本操作

1. **通貨ペアの選択**
   - BTC/JPY (ビットコイン)
   - ETH/JPY (イーサリアム)
   - XRP/JPY (リップル)
   - LTC/JPY (ライトコイン)

2. **売買シグナルの確認**
   - 画面中央の「売買シグナル」カードを確認
   - 推奨アクション(買い/売り/様子見)が表示されます
   - 詳細な分析結果と理由も確認できます

3. **通知の有効化**
   - 「通知を受け取る」トグルをON
   - 重要なシグナルが出たら自動で通知されます

## 🔧 技術仕様

### 使用API
- **コインチェック公開API**
  - エンドポイント: `https://coincheck.com/api/ticker`
  - 認証不要のパブリックAPI

### テクニカル指標の計算方法

#### RSI (Relative Strength Index)
```
RSI = 100 - (100 / (1 + RS))
RS = 平均上昇幅 / 平均下落幅
期間: 14
```
- 30以下: 売られすぎ(買いシグナル)
- 70以上: 買われすぎ(売りシグナル)

#### 移動平均線 (SMA)
- 短期: 7日間の単純移動平均
- 長期: 25日間の単純移動平均
- ゴールデンクロス(短期 > 長期): 買いシグナル
- デッドクロス(短期 < 長期): 売りシグナル

#### MACD
- EMA12 - EMA26
- プラス圏: 買いシグナル
- マイナス圏: 売りシグナル

### シグナル判定ロジック
各指標からスコアを計算し、総合的に判断:
- スコア +2以上: 買い推奨
- スコア -2以下: 売り推奨
- その他: 様子見

## 🚀 ローカル開発

### 必要なもの
- Webサーバー (Python、Node.js、Live Serverなど)

### 起動方法

#### Python を使う場合
```bash
cd crypto-advisor
python3 -m http.server 8000
```

#### Node.js (http-server) を使う場合
```bash
cd crypto-advisor
npx http-server -p 8000
```

ブラウザで `http://localhost:8000` を開く

### HTTPS対応
PWAの一部機能(通知、位置情報など)はHTTPSが必要です。
本番環境ではHTTPS対応のホスティングサービスを使用してください。
